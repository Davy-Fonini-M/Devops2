Objetivo do Projeto

Este projeto foi desenvolvido para demonstrar a implementação de um fluxo de trabalho DevOps completo, integrando práticas modernas de desenvolvimento e operações.

Tecnologias Utilizadas

  • Python Flask - Framework web leve e flexível
  • Docker - Containerização da aplicação
  • GitHub Actions - Automação de CI/CD
  • Pytest - Framework de testes automatizados

Fluxo de CI/CD

Nosso pipeline de integração e entrega contínua inclui:

  • Execução automática de testes a cada commit
  • Build e validação de imagens Docker
  • Implantação automatizada em ambientes de teste
  • Verificações de qualidade de código

Equipe

Este projeto é mantido por uma equipe dedicada de desenvolvedores e engenheiros DevOps comprometidos com as melhores práticas de desenvolvimento de software.
