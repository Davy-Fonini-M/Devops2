# Devops2
---------------
Criei uma página com estilo futurista como parte do projeto da disciplina de DevOps.
O objetivo foi aplicar na prática os conceitos de CI/CD, versionamento no GitHub e fluxo de pull requests. Além de aprender a automatizar o ciclo de desenvolvimento, aproveitei para explorar a criação de uma interface moderna e futurista, unindo design e prática de DevOps.
